este programa cria um arquivo SQL e usa o mesmo para fazer um dashboard que ajuda a ler o conteúdo dos CSV


o programa usa as bibliotecas pandas, dash, sqlalchemy e plotly na linguagem Python

instruções de uso:

na pasta arquivos:

-adicione aqui os 2 arquivos CSV do link https://www.kaggle.com/datasets/rankirsh/evolution-of-top-games-on-twitch


na pasta SQL:

-primeiro crie a conexão com o servidor MYSQL então use o arquivo "modelo_banco_de_dados.sql" para iniciar a database.

-após criada a database use "leitura_upload.ipynb" para adicionar as tabelas à database.


na pasta apresentação:

-execute "server.py" ele criará uma página no localhost, porta 8080 com os gráficos
